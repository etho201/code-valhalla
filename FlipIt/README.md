# FlipIt

## Two Dimentional Arrays

`int[][] twoDimentional = {{1,5},{2,6},{3,7},{4,8},{5,9}};`

Conceptual view:

|   | 0 | 1 |
|---|---|---|
| 1 | 1 | 5 |
| 2 | 2 | 6 |
| 3 | 3 | 7 |
| 4 | 4 | 8 |
| 5 | 5 | 9 |

Retrieving values:

| i | j |   | value: |
|---|---|---|:------:|
| 0 | 0 |   |    1   |
| 0 | 1 |   |    5   |
| 1 | 0 |   |    2   |
| 1 | 1 |   |    6   |
| 2 | 0 |   |    3   |
| 2 | 1 |   |    7   |
| 3 | 0 |   |    4   |
| 3 | 1 |   |    8   |
| 4 | 0 |   |    5   |
| 4 | 1 |   |    9   |
